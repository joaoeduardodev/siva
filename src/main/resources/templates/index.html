Como surgimos?

Um grande número de desempregados no país, surgiu a ideia de criarmos um sistema de vagas de emprego para as pessoas em uma posição desfavorecida na sociedade. Para criar um projeto com impacto social, uma equipe de desenvolvimento decidiu Criar o SIVA (Sistema de Vagas de Emprego), uma plataforma capaz de gerir as vagas de emprego e oferecer oportunidades de emprego para as pessoas.
